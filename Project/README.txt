Добрый день! Как запустить тестовое:

1) Вам понадобится Microsoft Visual Studiou Code, Webstorm или другая IDE

2) Открываете папку Project в IDE

3) Во вкладке Terminal:
	npm i node.js // ожидаете установки node.js ( если это необходимо ) 
	npm i json-server // приложение для имитации Базы Данных из json файла
	npm instal  // ожидаете установки зависимостей
	json-server --watch dataBase.json --port 4000 // запуск имитации БД

5) В новой вкладке Terminal:
	npm start  // в окне браузера должна отобразится страница
